# 需求创建

委托给 `requirement-creator` 代理执行。

## 触发场景

- "我想做..."、"我想开发..."
- "实现..."、"实现一个..."
- "创建一个新..."
- "添加功能..."
- "开始工作..."、"做一个新的..."
- 任何新任务/功能请求

## 执行

直接委托给 `requirement-creator` 代理处理所有创建工作。

## 禁止行为

- 在需求创建前开始规划
- 在需求创建前开始编码
- 跳过需求创建直接执行任何工作
