# 详细设计：需求进度追踪器文档标准化

## 基本信息

- **需求 ID**: req-20260329-222104
- **关联概要设计**: 概要设计.md
- **状态**: Draft → Review → Approved

## 1. 模块设计

### requirement-doc-writer

#### 职责
根据需求信息生成标准化的需求文档（PRD）

#### 公开接口
```
Agent: requirement-doc-writer
输入: 需求 JSON 文件路径
输出: .requirements/docs/{id}/需求文档.md
```

#### 内部结构
- Read 工具：读取需求 JSON 和模板文件
- Write 工具：生成需求文档

#### 实现逻辑
1. 读取需求 JSON 文件获取需求信息
2. 读取 requirements/templates/docs/requirement-doc-template.md 模板
3. 替换模板中的占位符为实际值
4. 创建 .requirements/docs/{id}/ 目录
5. 写入需求文档.md
6. 更新需求的 progressLog，记录 artifact

### design-doc-writer

#### 职责
生成概要设计和详细设计文档

#### 公开接口
```
Agent: design-doc-writer
输入: 需求文档路径 或 需求 JSON 文件路径
输出:
  - .requirements/docs/{id}/概要设计.md
  - .requirements/docs/{id}/详细设计.md
```

#### 内部结构
- Read 工具：读取需求文档和模板文件
- Write 工具：生成设计文档

#### 实现逻辑
1. 读取需求信息（从 JSON 或需求文档）
2. 读取两个模板文件
3. 替换模板占位符
4. 创建输出目录
5. 写入概要设计和详细设计
6. 更新 progressLog

## 2. 数据结构

### 核心类型定义

```javascript
// Agent 配置
{
  name: string,           // Agent 名称
  description: string,    // Agent 描述
  tools: string[],        // 可用工具
  model: string          // 使用的模型
}

// 需求文件结构
{
  id: string,             // 需求 ID
  name: string,           // 需求名称
  type: string,          // 需求类型
  status: string,        // 需求状态
  currentPhase: string,   // 当前阶段
  description: string,   // 描述
  acceptanceCriteria: string[],  // 验收标准
  progressLog: Array<{
    event: string,
    timestamp: string,
    details: object
  }>,
  progress: {
    phases: Array<{
      name: string,
      mode: string,       // 'manual' | 'auto'
      agent: string,      // Agent 名称
      status: string      // 'pending' | 'in_progress' | 'completed'
    }>
  }
}
```

## 3. 算法设计

### 文档生成算法

```
输入: 需求信息, 模板路径, 输出路径
输出: 生成的文档文件

1. 读取模板文件内容
2. 解析模板中的占位符 {{variable}}
3. 从需求信息中提取对应值
4. 替换所有占位符为实际值
5. 如果是 Handlebars 模板，编译并执行
6. 创建输出目录（如果不存在）
7. 写入文件
8. 返回输出路径
```

## 4. 错误处理

| 错误场景 | 错误码 | 处理方式 |
|----------|--------|----------|
| 模板文件不存在 | ENOENT | 抛出明确错误信息 |
| 输出目录创建失败 | EACCES | 报错并终止 |
| 需求文件不存在 | ENOENT | 抛出错误 |
| 模板解析失败 | EPARSE | 记录错误并使用 fallback |
